Part C — Debug

4 questions · 5 marks each · ~20 minutes. Find the bug, write the fix, explain in your own words (min 15 chars).

C1 (5 marks)
This Dog class is missing something — every method needs ONE special thing as its first parameter. Fix the __init__ line.
class Dog:
    def __init__(name):
        self.name = name

# TypeError when running Dog("Rex")
C2 (5 marks)
This Hero class doesn't actually store the hp. Fix the line inside __init__.
class Hero:
    def __init__(self, hp):
        hp = hp        # ← does nothing useful

h = Hero(100)
print(h.hp)   # AttributeError: 'Hero' object has no attribute 'hp'
C3 (5 marks)
This code prints something weird (the method object, not the result). Fix the last line.
class Hero:
    def __init__(self, hp):
        self.hp = hp

    def is_alive(self):
        return self.hp > 0

h = Hero(50)
print(h.is_alive)
# Prints: <bound method Hero.is_alive of ...>
# Should print: True
C4 (5 marks)
This take_damage method has TWO bugs: missing self as a parameter AND wrong attribute access. Fix the def line and the body line.
class Hero:
    def __init__(self, hp):
        self.hp = hp

    def take_damage(amount):     # ← bug 1
        hp -= amount             # ← bug 2

h = Hero(100)
h.take_damage(20)
