Part D — Coding Tasks (Write Python)

2 questions · 10 marks each · ~30 minutes. Write the code in VS Code, paste it below, then paste what it prints.

D1 (10 marks)
Task: Write a function called calculate_damage that takes hp, damage, and armour, and returns the new HP after the hit. Damage that gets through = damage - armour (but never below 0). The new HP is hp - damage_through.
Test cases your function must pass:
calculate_damage(100, 30, 10) → 80 (30 dmg, 10 armour soaks → 20 through → 100−20=80)
calculate_damage(100, 5, 50) → 100 (5 dmg, 50 armour soaks all → 0 through → 100−0=100)
calculate_damage(50, 70, 0) → −20 (70 dmg, no armour → all through → 50−70=−20)
D2 (10 marks) LIST COMPREHENSION + *args
Task: Write a function called average that takes any number of scores using *args, and returns their average. Then use a list comprehension to make a list of "high" scores (≥ 80) from a given list, and print the average of those high scores.
Expected behaviour:
average(80, 90, 100) → 90.0
Then for scores = [55, 92, 78, 88, 100, 67, 85]:
highs = [s for s in scores if s >= 80] → [92, 88, 100, 85]
average(*highs) → 91.25 (or similar — show what your code prints)
