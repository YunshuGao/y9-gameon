Numbers and Maths

Learning Intentions

  • I can do arithmetic in Python (+, -, *, /, **)
  • I can mix ints and floats
  • I can use the int() and str() functions to convert types

Success Criteria

  • My calculator.py asks for two numbers and prints their sum
  • I've used at least 3 different operators
  • My program doesn't crash on normal inputs
5 min

Do Now — Error of the Day

Predict the output: 

print(10 / 3)
print(10 // 3)
print(10 % 3)
print(2 ** 8)
Discuss with partner. Then run each in VS Code.

10 min

I Do — Live-coded demo

Live-code: demonstrate all 7 operators: +, -, *, /, //, %, **. Show input() returns a string: input("age: "). Show the trap: input() + 5 → TypeError. Fix with int(input()).

15 min

We Do — Guided build

Build calculator.py together: 

a = int(input("First number: "))
b = int(input("Second number: "))
print(f"Sum: {a + b}")
Extend to subtract, multiply, divide.

35 min

You Do — Your turn

Build game_stats.py: ask player for starting_health, damage_taken, potions_used (each +20 hp). Print final hp. If final hp < 0, print "You died 💀", else print health bar like HP: {hp}/100. Extension: damage_taken can be a float; use rounding in the print.

Support

calculator_starter.py with 2 operators pre-written; fill in the other 2.

Core

Independent game_stats.py build. Attempt Matthes Ch 2 exercises 2-5, 2-6.

Extension

Build a tax calculator that applies a 10% GST to a price; round to 2 decimals using round(x, 2).

5 min

Exit Ticket — Coding Journal

Journal: what's the difference between 10/3 and 10//3? Write it in your own words.

📝 Journal submission

Submit your 2-3 sentence journal entry in Google Classroom before you leave. Every entry contributes to your Coding Journal (/5) in the final assessment.
