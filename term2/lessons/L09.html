L09 · Week 5 ⏱ 75 minutes 🎯 IND5-2 · IND5-3 · IND5-7 📂 Phase 1 — Python Foundations

Classes — building your own objects

Learning Intentions

  • I can define a class with attributes and methods
  • I can create instances of a class
  • I understand what self means

Success Criteria

  • My Character class has at least 3 attributes + 2 methods
  • I've created at least 2 Character instances
  • I can call a method on an instance
5 min

Do Now — Error of the Day

Error of the Day — predict what happens: 

class Dog:
    name = "Rex"
    def bark(self):
        print(f"{self.name} says Woof!")
d = Dog()
d.bark()

10 min

I Do — Live-coded demo

Live-code a Character class: 

class Character:
    def __init__(self, name, hp):
        self.name = name
        self.hp = hp
    def take_damage(self, amount):
        self.hp -= amount
        print(f"{self.name} takes {amount} damage! HP: {self.hp}")

hero = Character("Arrow", 100)
hero.take_damage(20)
Explain __init__, self, attribute vs method.

15 min

We Do — Guided build

Extend Character together: add heal(amount), is_alive(), attack power attribute, attack(other) method that reduces the other's HP.

35 min

You Do — Your turn

Build character_class.py: your own Character class with at least: __init__, 3 attributes, 3 methods. Create 2 Character instances: a hero and an enemy. Have them take turns attacking until one dies.

Support

character_starter.py with __init__ pre-written; you add the methods.

Core

Complete character_class.py independently with hero-vs-enemy battle loop.

Extension

Subclass Character into Hero (with special_attack()) and Enemy (with AI that picks attack type randomly).

5 min

Exit Ticket — Coding Journal

Journal: in one sentence, what's the difference between the Character CLASS and a Character INSTANCE?

Submit your 2-3 sentence journal entry in Google Classroom before you leave. Every entry contributes to your Coding Journal (/5) in the final assessment.
