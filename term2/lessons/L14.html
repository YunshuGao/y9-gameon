Aliens appear!

Learning Intentions

  • I can create an Alien class similar to Ship
  • I can draw a single alien on screen
  • I can read pygame docs to find new functions

Success Criteria

  • One alien appears in the upper-left of my screen
  • I understand how it relates to my Ship class
  • My alien uses its own image (alien.bmp)
5 min

Do Now — Error of the Day

Download alien.bmp from the hub. Save in your game folder.

10 min

I Do — Live-coded demo

Live-code Alien class — a near copy of Ship: 

class Alien(pygame.sprite.Sprite):
    def __init__(self, screen):
        super().__init__()
        self.screen = screen
        self.image = pygame.image.load('alien.bmp')
        self.rect = self.image.get_rect()
        self.rect.x = self.rect.width
        self.rect.y = self.rect.height
    
    def blitme(self):
        self.screen.blit(self.image, self.rect)
Emphasise: we're reusing what we know from Ship. Noticing the pattern is the key programming skill.

15 min

We Do — Guided build

Integrate: create alien = Alien(screen) in main. Call alien.blitme() in the draw section. Move the alien: set alien.rect.x = 100, alien.rect.y = 100.

35 min

You Do — Your turn

Create 3 aliens at different positions. Put them in a group: aliens = pygame.sprite.Group(). Iterate over the group with for alien in aliens: alien.blitme(). Extension: replace alien.bmp with your own design.

Support

Use alien_class_starter.py. Make 1 alien appear.

Core

3 aliens at different positions using a sprite group.

Extension

Use nested for-loops to create a 5x3 grid of aliens.
