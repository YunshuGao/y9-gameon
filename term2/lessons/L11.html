Installing Pygame + your first window

Learning Intentions

  • I can install Pygame using pip
  • I can open a blank Pygame window
  • I understand the game loop

Success Criteria

  • Pygame installed — import pygame works
  • My alien_invasion.py opens a 1200x800 blue window
  • The window closes properly when I click the X
5 min

Do Now — Error of the Day

Open a terminal. Type pip --version. If it says a version number, 👍. If not, raise hand.

10 min

I Do — Live-coded demo

Live-demo: install Pygame with pip install pygame. Create alien_invasion.py. Type the minimal Pygame skeleton from Matthes Ch 12 live: 

import sys
import pygame

def run_game():
    pygame.init()
    screen = pygame.display.set_mode((1200, 800))
    pygame.display.set_caption("Alien Invasion")
    bg_color = (230, 230, 230)
    
    while True:
        for event in pygame.event.get():
            if event.type == pygame.QUIT:
                sys.exit()
        screen.fill(bg_color)
        pygame.display.flip()

run_game()

15 min

We Do — Guided build

Class types along. Run it. See the window appear. Now experiment: change the (1200, 800) size. Change the background colour. Talk about RGB tuples — (230, 230, 230) is light grey, (0, 0, 255) is pure blue.

You Do — Your turn

Make the window YOUR way: choose a window size (600x400 minimum), a custom title, and a background colour that isn't grey. Experiment with 3 different colours and pick the best. Extension: create a Settings class holding screen_width, screen_height, bg_color.
