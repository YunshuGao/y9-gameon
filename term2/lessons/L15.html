L15 · Week 8 ⏱ 75 minutes 🎯 IND5-2 · IND5-3 · IND5-7 📂 Phase 3 — Alien Invasion Mechanics

The alien fleet — rows and columns

Learning Intentions

  • I can use nested for-loops to create a grid of aliens
  • I can calculate how many aliens fit on screen
  • I can create a _create_fleet() helper method

Success Criteria

  • My screen shows a proper fleet (5+ columns × 2+ rows)
  • The fleet is centred with even spacing
  • I understand how the loop creates each alien
5 min

Do Now — Error of the Day

Predict: for row in range(3): for col in range(5): print(row, col). How many lines of output?

10 min

I Do — Live-coded demo

Live-code fleet creation method: 

def _create_fleet(self):
    alien = Alien(self.screen)
    alien_width = alien.rect.width
    available_space_x = self.settings.screen_width - 2 * alien_width
    number_aliens_x = available_space_x // (2 * alien_width)
    
    for alien_num in range(number_aliens_x):
        alien = Alien(self.screen)
        alien.x = alien_width + 2 * alien_width * alien_num
        alien.rect.x = alien.x
        self.aliens.add(alien)
Walk through the maths: why divide by 2 * alien_width? (Alien + gap.)

15 min

We Do — Guided build

Class types along. Extend to multiple rows: add outer for row_num in range(number_rows):. Calculate number_rows based on screen height. See the fleet spawn.

35 min

You Do — Your turn

Tune your fleet: pick a column count and row count that looks good. Extension: different alien colours per row (use tinted sprites or draw pygame.draw.rect). Extension 2: fleet starts offset 50px from the top.
