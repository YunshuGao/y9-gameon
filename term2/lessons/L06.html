L06 · Week 3 ⏱ 75 minutes 🎯 IND5-2 · IND5-3 · IND5-5 📂 Phase 1 — Python Foundations

If statements — making decisions

Learning Intentions

  • I can use if / elif / else to branch my program
  • I can compare values with ==, !=, <, >, <=, >=
  • I can combine conditions with and / or / not

Success Criteria

  • My age_check.py asks for age and gives different responses
  • My text adventure v1 has at least 3 branches
  • My code uses at least one elif
5 min

Do Now — Error of the Day

Error of the Day: if age = 18: print("adult") — why doesn't this work? (Answer: = is assignment, == is comparison.)

10 min

I Do — Live-coded demo

Live-code: 

score = int(input("Enter score: "))
if score >= 85:
    print("Grade A")
elif score >= 70:
    print("Grade B")
elif score >= 50:
    print("Grade C")
else:
    print("Needs improvement")

15 min

We Do — Guided build

Build the start of a text adventure together: adventure_v1.py. 

print("You stand at a crossroads.")
choice = input("Go left or right? ")
if choice == "left":
    print("You find treasure! 🏆")
elif choice == "right":
    print("You fall into a pit 💀")
else:
    print("You stand still, forever.")

35 min

You Do — Your turn

Extend adventure_v1.py: add 2 more rooms, each with a choice and 3 branches. Aim for at least 6 outcomes total. Use f-strings so the player's name appears in the story.

Support

adventure_v1_starter.py with the first room pre-written; you add 1 more.

Core

3+ rooms, 6+ outcomes, player name included.

Extension

Add a 'health' integer: some choices reduce health. If health ≤ 0, print game-over message.

5 min

Exit Ticket — Coding Journal

Journal: paste one if statement from your code. Explain in one sentence what it does.

📝 Journal submission

Submit your 2-3 sentence journal entry in Google Classroom before you leave. Every entry contributes to your Coding Journal (/5) in the final assessment.
