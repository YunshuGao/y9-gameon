Lesson 5: Pygame Setup + First Window (Matthes Ch 12 §1-2)

Learning Intentions

  • Install Pygame, open a game window, understand the game loop pattern.

Success Criteria

  • Pygame installed on my laptop.
  • My alien_invasion.py opens a navy-coloured window.
  • The window closes cleanly when I click X.

📍 Do Now — Error of the Day

Type pip --version in your terminal. What comes up? (This checks if pip is installed — it comes free with Python.)

👁 I Do — Teacher Demo

Live-install pygame (pip install pygame), then live-code the minimal 20-line Pygame skeleton: init, display.set_mode, event loop, fill, flip. Explain what each line does.

🤝 We Do — Build Together

Class types along — everyone gets a Pygame window open. Verify with a fist-bump check: raise hand if your window opens. We troubleshoot live.

🚀 You Do — Your Turn

Customise your window: size, background colour, title. Try 3 different colour schemes. Extension: refactor into a Settings class. Arcade quests below cover the basics.

  • pip install pygame (troubleshoot 1:1)
  • Type the minimal Pygame skeleton from scratch
  • Change window size + background colour
  • Understand what each of pygame.init(), display.set_mode, event.get, display.flip does
