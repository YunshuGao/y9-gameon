L05 · Week 3 ⏱ 75 minutes 🎯 IND5-3 · IND5-7 📂 Phase 1 — Python Foundations

Loops — making Python repeat itself

Learning Intentions

  • I can use a for-loop to iterate over a list
  • I can use range() to repeat actions a fixed number of times
  • I can combine a loop with an f-string

Success Criteria

  • My times_table.py prints a multiplication table
  • I've used at least 2 for-loops today
  • I understand what range(1, 5) produces
5 min

Do Now — Error of the Day

On the board: for i in range(3): print(i). Predict the output. Does it print 0,1,2 or 1,2,3? Partner-discuss.

10 min

I Do — Live-coded demo

Live-code: 

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like {fruit}")
Emphasise: indentation matters in Python! The 4 spaces tell Python what's inside the loop. Show what happens without indentation — IndentationError.

15 min

We Do — Guided build

Build multiplication.py together: 

number = 7
for i in range(1, 11):
    print(f"{number} x {i} = {number * i}")
Try different numbers. Try nested loops briefly (warn: this is hard!).

35 min

You Do — Your turn

Build stats_report.py: a list of 5 players with scores (use tuples or parallel lists). Loop through and print each one as Player 1: Alice - Score: 87. Use enumerate() to get both index and value. Extension: calculate and print the class average using sum() and len().

Support

Pair with a Core-tier partner; they drive first 15 min, you drive last 15 min.

Core

Complete stats_report.py. Run Matthes Ch 4 exercises 4-1, 4-3.

Extension

Nested loop challenge: print a 10x10 multiplication table using nested for-loops.

5 min

Exit Ticket — Coding Journal

Journal: what does Python use to tell where a loop starts and ends? (Hint: it's not { })

📝 Journal submission

Submit your 2-3 sentence journal entry in Google Classroom before you leave. Every entry contributes to your Coding Journal (/5) in the final assessment.
