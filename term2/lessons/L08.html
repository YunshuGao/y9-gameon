Lesson 8: Bullet Class + Sprite Inheritance (Matthes Ch 12 §5)

Learning Intentions

  • Build a Bullet class that inherits from pygame.
  • sprite.
  • Sprite.

Success Criteria

  • My Bullet class inherits from Sprite using class Bullet(Sprite).
  • I can explain what super().__init__() does.
  • A single bullet can be fired and drawn.

📍 Do Now — Error of the Day

Look at this line: class Bullet(pygame.sprite.Sprite): — what does it mean? Discuss with a partner for 1 minute.

👁 I Do — Teacher Demo

Live-code the Bullet class with Sprite inheritance + super().__init__(). Draw a diagram: Sprite = parent class, Bullet = child class. Child gets everything parent has, plus extras.

🤝 We Do — Build Together

Class builds the Bullet class together. Create ONE test bullet and draw it on screen at ship's midtop. Verify: everyone sees a bullet.

🚀 You Do — Your Turn

Finish the Bullet class with update + draw_bullet methods. Customise colour, size. Arcade quests below cover Sprite inheritance + rect basics.

  • Import Sprite from pygame.sprite
  • Build Bullet class with super().__init__()
  • Create Rect at ship.rect.midtop
  • Write update() and draw_bullet() methods
