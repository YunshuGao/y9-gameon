L01 · Week 1 ⏱ 75 minutes 🎯 IND5-3 · IND5-9 📂 Phase 1 — Python Foundations

Hello Python — setting up your editor

Learning Intentions

  • I can install Python and VS Code on my laptop
  • I can run my first Python script
  • I can use print() to display a message

Success Criteria

  • Python 3.12+ is installed on my laptop
  • VS Code opens and shows a .py file
  • I have saved hello.py and it prints my name
5 min

Do Now — Error of the Day

Open a blank document and answer in 2-3 lines: what do you think a "programming language" is? Compare it to English. You'll compare answers with a partner in 3 minutes.

10 min

I Do — Live-coded demo

Teacher live-installs Python from python.org, installs VS Code, and opens it. Writes print("Hello, Year 9!") on screen. Runs it in the terminal. Shows what happens when you mis-spell print — introduces NameError. Point out: Python is literal. Every character matters.

15 min

We Do — Guided build

Class follows along at own laptops: install Python (teacher helps each student individually — this is the moment nobody can be left behind). Test by opening Command Prompt / Terminal and typing python --version. Install VS Code. Open it. Create hello.py. Type print("Hello, my name is [YOUR NAME]!"). Run it.

35 min

You Do — Your turn

Goal: write a 3-line program that prints a poster introducing yourself. Requirements: one line with your name, one line with your favourite game, one line with what you hope to learn. Save it as intro.py. Extension: use print() with no arguments to add blank lines for layout. Extension 2: add ASCII art (search 'ascii art' — try a gamer pattern).

Support

Teacher pair-programs with you side-by-side for install. You follow along on your own screen. Pre-typed template intro.py with TODO markers.

Core

Follow this tutorial independently. Complete intro.py with 3 working print statements.

Extension

Add 5+ print statements, use escape characters (\n, \t). Try print("text", end="") to control line endings.

⚠ WHS Reminder

First lesson at laptops — check posture and screen brightness. Take a 30-second eye-rest break at 35 min.

5 min

Exit Ticket — Coding Journal

Open Google Classroom. In the Journal assignment, write 3 sentences: (1) Did Python install work? (2) What did you print in your intro.py? (3) What was one thing that surprised you today?

📝 Journal submission

Submit your 2-3 sentence journal entry in Google Classroom before you leave. Every entry contributes to your Coding Journal (/5) in the final assessment.
