L07 · Week 4 ⏱ 75 minutes 🎯 IND5-3 · IND5-5 · IND5-7 📂 Phase 1 — Python Foundations

User input and while loops — interactive games

Learning Intentions

  • I can use input() to get user choices
  • I can use a while-loop to keep asking until something happens
  • I can use a flag variable to exit a loop cleanly

Success Criteria

  • My adventure v2 loops until the player quits or dies
  • I've used a flag variable (e.g. alive = True)
  • My program handles at least one wrong input
5 min

Do Now — Error of the Day

Spot the bug: 

count = 0
while count < 5:
    print(count)
Why does this run forever? How do you fix it?

10 min

I Do — Live-coded demo

Live-code: guess-the-number game. Computer picks a secret; player guesses until correct. 

import random
secret = random.randint(1, 100)
guess = 0
while guess != secret:
    guess = int(input("Guess (1-100): "))
    if guess < secret: print("Higher!")
    elif guess > secret: print("Lower!")
print("You got it!")
