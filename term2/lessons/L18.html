Lesson 18: Multiple Alien Types + Boss Alien

Learning Intentions

  • Stop all aliens being the same.
  • Top row = fast/weak.
  • Middle = standard.

Success Criteria

  • My fleet has 3 visually-different alien types.
  • One alien type takes 3 hits to destroy.
  • Destroying a BOSS alien awards 500 points.

📍 Do Now — Error of the Day

Look at your current alien fleet. Every alien is identical. That's a design choice — do you love it, or does it feel flat?

👁 I Do — Teacher Demo

Live-code subclassing: class FastAlien(Alien), class TankAlien(Alien), class BossAlien(Alien). Override self.hp, self.points, the image path.

🤝 We Do — Build Together

Modify _check_bullet_alien_collisions to use alien.hit() instead of auto-killing. Spawn a mixed fleet: row 0 = Fast, row 1 = Standard, row 2 = Tank.

🚀 You Do — Your Turn

Draw 3 distinct alien sprites (or use the pack I'll share in Classroom). Add the BossAlien subclass. Playtest — does the fleet feel more strategic now?

  • Add self.hp = 1 to Alien.__init__
  • Subclass FastAlien(Alien): self.hp = 1; self.speed_factor = 2
  • Subclass TankAlien(Alien): self.hp = 3; load red sprite
  • Subclass BossAlien(Alien): self.hp = 10; larger sprite
  • In _check_bullet_alien_collisions: decrement hp instead of kill; only kill when hp == 0
