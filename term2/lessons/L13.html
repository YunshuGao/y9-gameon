L13 · Week 7 ⏱ 75 minutes 🎯 IND5-3 · IND5-5 📂 Phase 2 — Pygame Foundations

Firing bullets

Learning Intentions

  • I can create a Bullet class
  • I can manage a group of bullets with pygame.sprite.Group
  • I can fire bullets with the spacebar

Success Criteria

  • Pressing spacebar fires a bullet upward from my ship
  • Bullets are removed when they leave the top of the screen
  • I can hold spacebar and multiple bullets stay on screen
5 min

Do Now — Error of the Day

Think: when the ship fires, the bullet needs to START at the ship's position. Where on the ship should it come from? (Top middle.)

10 min

I Do — Live-coded demo

Live-code Bullet class inheriting from pygame.sprite.Sprite: 

class Bullet(pygame.sprite.Sprite):
    def __init__(self, screen, ship):
        super().__init__()
        self.screen = screen
        self.rect = pygame.Rect(0, 0, 3, 15)
        self.rect.midtop = ship.rect.midtop
        self.color = (60, 60, 60)
        self.speed = 1
    
    def update(self):
        self.rect.y -= self.speed
    
    def draw_bullet(self):
        pygame.draw.rect(self.screen, self.color, self.rect)

15 min

We Do — Guided build

In the main game, create bullets = pygame.sprite.Group(). On KEYDOWN K_SPACE, bullets.add(Bullet(screen, ship)). In the main loop, call bullets.update() and iterate to draw each. Clean up: remove bullets that leave the screen with if bullet.rect.bottom <= 0: bullets.remove(bullet).

35 min

You Do — Your turn

Tune your bullets: width, height, colour, speed. Try a colour that matches your ship. Extension: add a limit (max 3 bullets on screen at once); add cooldown (1 bullet per 200ms).

Support

Use bullet_class_starter.py with structure pre-written.

Core

Complete bullet class per Matthes Ch 12.

Extension

Add bullet types — slow wide bullet (shotgun) vs fast narrow bullet (laser). Use class inheritance.

5 min
