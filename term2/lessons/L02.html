Variables and Strings

Learning Intentions

  • I can create a variable and store a value in it
  • I can use f-strings to combine text and variables
  • I can use string methods like .upper() and .lower()

Success Criteria

  • My program stores at least 3 variables
  • I have used an f-string to insert a variable into output
  • My output is formatted nicely
5 min

Do Now — Error of the Day

Fix this broken code (it's on the board): print(Hello) — why does Python error? Discuss with a partner for 2 min. (Answer: Python looks for a variable called Hello, doesn't find one.)

10 min

I Do — Live-coded demo

Live-code: create variables name = "Yunshu", age = 30. Show print(name). Demonstrate the critical distinction between "Yunshu" (string) and Yunshu (variable). Introduce f-string: print(f"Hi, I am {name} and I am {age} years old").

15 min

We Do — Guided build

Students open madlib.py. Together we build a 3-variable mad-lib: 

noun = "banana"
adjective = "spicy"
verb = "dancing"
print(f"The {adjective} {noun} is {verb}!")
Run it. Change variable values. Run again. See how the output changes without changing the print.

35 min

You Do — Your turn

Build gamer_profile.py: 5 variables (player name, character class, hp, attack, favourite weapon), all printed using f-strings in a nice format. Try .upper() on the name (e.g. name.upper()). Extension: add string concatenation with + and compare with f-strings — which is cleaner?

Support

Starter file madlib_starter.py with TODO markers. Pair with a Core-tier partner.

Core

Complete gamer_profile.py independently. Attempt 1-2 of Matthes Ch 2 exercises as homework.

Extension

Explore string methods: .title(), .strip(), .replace(). Build a caesar cipher: shift each letter by 3.

5 min

Exit Ticket — Coding Journal

Journal: which f-string did you write that you're proudest of? Paste it into the journal entry.

📝 Journal submission

Submit your 2-3 sentence journal entry in Google Classroom before you leave. Every entry contributes to your Coding Journal (/5) in the final assessment.
