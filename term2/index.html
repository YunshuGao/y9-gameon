What you'll build

Over 11 weeks, you will learn Python programming from scratch and build one complete arcade game — Alien Invasion style — with your own original twist. This is a real programming course disguised as a game project. Every concept you learn has an immediate use in the game.

Phase 1 · Weeks 1-5

Python fundamentals — variables, loops, if, functions, classes. Build a text adventure.

Phase 2-3 · Weeks 6-9

Pygame — ship, bullets, aliens, fleets, collisions, game over.

Phase 4 · Weeks 10-11

Polish, Your Twist, playtest, showcase.
