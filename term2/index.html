What you'll build

Over 11 weeks, we build ONE complete arcade game — Alien Invasion — together. I walk through every class, every line, every design decision on the projector and you code along with me. Final project: a personal reskin of your working game + a 500-word evaluation.

Phase 1 · Weeks 1-2

Python refresher (you had Python in Y8) — diagnostic + loops + functions + classes.

Phase 2 · Weeks 3-7

Alien Invasion core — ship, bullets, sprite groups, fleets, collisions, lives, scoring.

Phase 3-4 · Weeks 8-11

Extensions — difficulty, high scores, sfx + particles, alien variants, power-ups, custom art, reskin + showcase.
