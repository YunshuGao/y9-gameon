Kenney Asset Guide

Which files to use for each Unity lesson

← Back to Tutorial Hub
What is this? This guide tells you exactly which files from the Kenney Platformer Art Deluxe pack to import for each lesson. Each page shows the file names, where to find them, how to set them up in Unity, and what they should look like in your scene.

Rule of thumb: Only import what you need for the current lesson. You can always add more later!

Asset pack licence: CC0 (Public Domain) — free for any use, no credit required.
