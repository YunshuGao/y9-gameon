# Y9 Game On — Arthur Phillip High School

Teaching hub for Year 9 Game On (9GAMZA) elective at Arthur Phillip High School, NSW.

Live at: **https://yunshugao.github.io/y9-gameon/**

## Structure

- `index.html` — landing page; links to Term 1 + Term 2
- Term 1 content (2D game dev in Unity + Scratch) lives at repo root
- `term2/` — Term 2 hub: Python + Pygame, 22 lesson pages + checkpoints + exemplars

## Term 2 highlights

- 22 lesson pages with full 75-min 5-segment structure (Do Now / I Do / We Do / You Do / Exit)
- 3-tier differentiation (Support / Core / Extension) on every lesson
- Checkpoint .py files at Lessons 11, 14, 17, 19 so stalled students can re-enter
- Your Twist ideas page (10 options from easy to very hard)
- Coding Journal template page with example entry
- Full Alien Invasion reference game (`term2/exemplars/alien_invasion_complete.py`)

## Assessment

One task, /25 (25% weighting), due Week 11 Lesson 22:
- Coding Journal (/5)
- Python Alien Invasion game (/10)
- Code design & commenting quality (/5)
- Evaluation — 500-word written OR 2-min video (/5)

## Syllabus

NSW NESA Industrial Technology Stage 5 (IND5-2, IND5-3, IND5-5, IND5-7, IND5-9).
